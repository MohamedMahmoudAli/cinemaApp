# Cinema App

## Description
A Flutter application for browsing movies and playing external video streams.

## Features
- Browse movies and series
- Video playback
- Multiple qualities

## Video Player
Plays MP4 and HLS streams.

### Known Issues
Some video links may not work due to server or format limitations.

## Getting Started

### Installation
1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`

## Disclaimer
This app does not host any content.
